## Features

- Bulk image upload → ZIP download
- Single image → direct download
- Fit or crop to 800x800 canvas
- White or Transparent background
- Optional background removal (requires rembg)
- Supports PNG, WEBP, JPG formats
- Safe filenames and network ready
